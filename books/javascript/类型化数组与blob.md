# 类型化数组与blob
## 类型化数组
### 与数组的区别
- 类型化数组的元素都是数字，使用构造函数在创建类型化数组的的时候决定了决定了数组中数字的类型（无符号或有符号整数，浮点数）和大小。
- 类型化数组有固定的长度
- 在创建类型化数组的时候，数组中的元素总是默认值为0
### 构造函数
```js
  Int8Array() // 有符号字节
  Uint8Array() // 无符号字节
  Int16Array() // 有符号16位短整数
  Uint16Array() // 无符号16位短整数
  Int32Array() // 有符号32位整数
  Uint32Array() // 无符号32位整数
  Float32Array() // 32位浮点数值
  Float64Array() // 64位浮点数值
```
